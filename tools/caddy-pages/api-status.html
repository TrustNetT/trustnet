Core Endpoints

GET /blocks
/cosmos/base/tendermint/v1beta1/blocks/latest
Get latest block information
Example
curl -s https://api.trustnet.local/cosmos/base/tendermint/v1beta1/blocks/latest | jq .
GET /bank/balances
/cosmos/bank/v1beta1/balances/{address}
Get account balances
Example
curl "https://api.trustnet.local/cosmos/bank/v1beta1/balances/cosmos1..." | jq .
GET /staking/validators
/cosmos/staking/v1beta1/validators
Get list of validators
Example
curl -s https://api.trustnet.local/cosmos/staking/v1beta1/validators | jq .
POST /broadcast/tx
/cosmos/tx/v1beta1/txs
Broadcast a transaction
Example
curl -X POST https://api.trustnet.local/cosmos/tx/v1beta1/txs -d '...' -H "Content-Type: application/json"

Authentication

This API endpoint is open and does not require authentication for read operations. For write operations (transactions), you'll need to sign the transaction with your private key.
