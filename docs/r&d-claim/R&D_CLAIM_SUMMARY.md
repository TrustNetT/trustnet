### Before Submitting to HMRC
1. Complete time-tracking totals
2. Estimate total man-hours × hourly rate
3. Add infrastructure costs
4. Compile all evidence into organized folders
5. Export GitHub commit history (use `git log` with custom format)
6. Take screenshots of key technological achievements

### HMRC Submission
- Use government's free online form or HowDo.app
- Upload this summary + supporting documentation
- Reference GitHub evidence: commit hashes, file changes
- Submit with tax return (April 2027)
- Keep all evidence for 6 years in case of HMRC inquiry
