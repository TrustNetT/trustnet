# TrustNet YouTube - Quick Start Checklist

## ✅ Phase 1: Account Setup (Do this FIRST)

### Step 1: Create Google Account (5 minutes)
- [ ] Go to https://accounts.google.com
- [ ] Click "Create account" 
- [ ] Choose business account setup
- [ ] Use email: hello@trustnet.io
- [ ] Verify phone & email
- [ ] **Status**: ✅ Ready

### Step 2: Create YouTube Channel (5 minutes)
- [ ] Go to https://youtube.com
- [ ] Sign in with Google account you just created
- [ ] Click profile → "Create a channel"
- [ ] Channel name: **TrustNet**
- [ ] Click "Create"
- [ ] **Status**: ✅ Channel now exists at youtube.com/@TrustNet

### Step 3: Basic Channel Info (10 minutes)
- [ ] Channel Settings → Basic Info
- [ ] Channel Name: TrustNet
- [ ] Description: Copy from YouTube-Channel-Launch-Guide.md (Phase 2)
- [ ] Channel URL: Claim custom URL @TrustNet
- [ ] **Save**
- [ ] **Status**: ✅ Basic setup complete

---

## 📋 Phase 2: Branding (Do this WEEK 1)

### Design Assets Needed
- [ ] **Profile Picture** (800×800px)
  - [ ] Source: Extract TrustNet logo from existing materials
  - [ ] OR design a clean, simple logo
  - [ ] Upload to: Settings → Branding → Channel icon
  
- [ ] **Banner** (2560×1440px)
  - [ ] Design using Canva (free template)
  - [ ] Text: "One Person. One Identity. One Protocol."
  - [ ] Colors: Teal (#00A8CC), Dark Blue (#0066CC)
  - [ ] Upload to: Settings → Branding → Banner

- [ ] **Intro/Outro Video** (3-5 seconds each)
  - [ ] Create in: Canva or CapCut
  - [ ] Include: TrustNet logo + "Subscribe!" text
  - [ ] Save as template for every video

### Channel Customization
- [ ] Settings → Layout → Enable "Sections"
- [ ] Create sections:
  - [ ] Latest Videos (auto)
  - [ ] Free Speech Series
  - [ ] TrustNet Pitch
  - [ ] Educational Content

- [ ] Settings → Monetization → Join YouTube Partner Program (when eligible)

---

## 🎬 Phase 3: Content Creation (WEEK 2-3)

### Script First Video
- [ ] Title: "What is TrustNet? (3-minute intro)"
- [ ] Script length: 300-400 words (3 min ~100 words/min)
- [ ] Structure:
  - [ ] Hook: "Big Tech controls your voice. We're fixing that."
  - [ ] Problem: "14.5M children can't verify their identity online"
  - [ ] Solution: "TrustNet: One person. One identity. One protocol."
  - [ ] CTA: "Subscribe to learn why this matters"

### Record First Video
- [ ] Set up OBS Studio (download free from obsproject.com)
- [ ] Test audio (USB mic or built-in)
- [ ] Record screen + webcam for 3 minutes
- [ ] Re-record if mistakes (aim for 1-2 takes)

### Edit First Video
- [ ] Download DaVinci Resolve (free from davinciresolve.com)
- [ ] Import video file
- [ ] Add intro/outro (3 sec each)
- [ ] Add captions (auto-generated or manual)
- [ ] Color grade (match TrustNet colors)
- [ ] Add music (30 sec from YouTube Audio Library)
- [ ] Export as MP4 (1080p)

### Create Thumbnail
- [ ] Use Canva (free version)
- [ ] Dimensions: 1280×720px
- [ ] Template: Black background + teal text + "What is TrustNet?"
- [ ] Save as PNG

---

## 🚀 Phase 4: First Upload (WEEK 3)

### Upload to YouTube
- [ ] Click "Create" → "Upload video"
- [ ] Select your MP4 file
- [ ] Wait for upload to complete

### Fill in Video Details
- [ ] **Title**: "What is TrustNet? | 3-Minute Intro"
- [ ] **Description**: Use template from Launch Guide
- [ ] **Thumbnail**: Upload your custom thumbnail
- [ ] **Tags**: trustnet, identity, blockchain, free speech, decentralization
- [ ] **Category**: Education or Tech
- [ ] **Chapters**: Yes (if video is 10+ min)
- [ ] **Premiere settings**: Schedule premiere for Friday 5pm (engagement)

### Visibility & Publishing
- [ ] Visibility: "Public" (not Private/Unlisted)
- [ ] Click "Schedule" or "Publish"
- [ ] **Status**: ✅ First video is LIVE!

---

## 📢 Phase 5: Promotion (DAY OF UPLOAD)

### Immediate Actions
- [ ] Share on LinkedIn (tag yourself, use free speech content)
- [ ] Share on Twitter (use @TrustNet handle)
- [ ] Share in Discord (if you have one)
- [ ] Send email to contacts: "First TrustNet video live!"
- [ ] Pin video link to any website/landing pages

### Engagement
- [ ] Reply to ALL comments in first 24 hours
- [ ] Thank people for subscribing (pin comment)
- [ ] Ask question to encourage comments ("What do you think about free speech?")

## 🎥 Phase 6: Content Pipeline (WEEKS 4+)

### Batch Recording (More Efficient)
**Instead of**: Recording 1 video at a time  
**Do this**: Record 4-5 videos in one session

- [ ] Record all intros together (consistency)
- [ ] Record main content videos back-to-back
- [ ] Record all outros together
- [ ] Stack recording session = 1 day of edits = faster pipeline

### Publishing Schedule
- [ ] **Tuesday**: Pitch breakdown or educational video
- [ ] **Friday**: Free speech article series video
- [ ] Repeat for 10 weeks (covers both series)

### Video Sequence (Recommended Order)
1. Intro to TrustNet (DONE)
2. "The Illusion of Free Speech" (Free Speech #1)
3. TrustNet Pitch Breakdown (Slide breakdown #1)
4. "Who Controls Your Voice?" (Free Speech #2)
5. ... continue pattern
