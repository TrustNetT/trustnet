## VERIFIED TIER 0: Government Grants + Non-Dilutive Funding (Priority 🎯)

### 1. R&D Tax Relief (UK Government)

**What it is**: Cash rebate on R&D spending (government reimburses you)
**Amount**: 20-33% of eligible R&D spending
**For TrustNet**: Blockchain development + cryptography = 100% qualifies
**Timeline**: Apply after 9 months of trading (you're ready now)
**Equity Cost**: ZERO (free government money, no dilution)

**Eligible Costs**:
- Developer salaries (% allocatable to R&D)
- Cloud infrastructure (AWS, etc.)
- Security audits
- Blockchain node development
- Ed25519 crypto implementation

**Example Calculation**:
- Year 1 R&D spend: £50,000 (developers, cloud, security)
- Tax relief @ 25%: £12,500 cash rebate
- This is money that comes BACK to your company

**How to Apply** (DIY or with Help):

**Option A: Do It Yourself (FREE)**
1. Document all R&D activities NOW (while doing the work):
   - GitHub commit logs (proof of development)
   - Security audit reports
   - Development journal/notes (what problems you solved, why)
   - Time spent per category (coding, testing, problem-solving)
2. Keep records organized:
   - Folder structure: `./docs/r&d-claim/` with dated evidence
   - Screenshot GitHub commit history quarterly
   - Save any security audit reports or testing documentation
3. When filing taxes (April):
   - Use HMRC's online form or free software
   - Report estimated R&D spend + hours
   - Attach documentation as evidence
   - HMRC may request clarification (respond with your GitHub logs)
4. You handle correspondence directly with HMRC
5. Receive cash rebate within 3-6 months

**Option B: Hybrid (Later, After Funding)**
1. Do preliminary DIY documentation (above)
2. After securing Tier 0 funding or accelerator funding
3. THEN hire accountant (£500-1,500) to optimize claim + handle follow-ups
4. Accountant likely recovers their fee through maximizing deductions

**Option C: Use Free/Cheap Software (DIY++)**
- **HowDo**: Free R&D tax relief software (does documentation for you)
- **Artemis**: Low-cost software (£50-100) that guides you through claim
- Both handle HMRC submission automatically

**Key Principle**: You DON'T need an accountant to START this process. Accountants help if you want professional optimization/audit protection, but DIY is totally valid and common for pre-funding startups.

**Recommended**: Do Option A (DIY documentation) NOW. After you have runway from funding, hire accountant to handle Year 2+ claims.

**Resources**: 
- UK government guide: https://www.gov.uk/guidance/corporation-tax-relief-for-spending-on-research-and-development
- HMRC R&D relief: https://www.gov.uk/topic/business-tax/research-and-development-relief

**Action**: Hire accountant THIS MONTH (3-week turnaround for claim preparation)

**Step 4: Choose Best Fit**

- Enter accelerator programme (13 weeks typical)

**Step 5: Demo Day (End of Accelerator)**
- Pitch to 100+ investors
- Generate seed round conversations

### For Seed Funds (Longer Sales Cycle - 4-8 months)

**Step 1: Prepare Same Materials** (as above)

**Step 2: Get Warm Introduction**
- Ask accelerator mentors for introductions
- Leverage your network (people who know fund principals)
- Cold outreach as backup

**Step 3: Schedule 1:1s**
- Typical: 20-30 minute initial call
- If promising: Follow-up meetings (2-3 total)
- Due diligence phase if serious

**Step 4: Term Sheet**
- If fund interested, they'll send term sheet
- Negotiate terms (amount, equity %, board seat, etc.)
- Sign and close
